# CordyChecker
CordyChecker — программа, предназначенная для ускорения процесса проверки с сохранением высокой точности результатов. Создана с целью облегчить работу модераторов и повысить эффективность выявления читеров, даже с использованием сложных программных средств, которые могут быть спрятаны.

# Основные особенности
Ускорение проверок: CordyChecker ускоряет процесс проверки, при этом не снижая его качества.
Безопасность: Программа была проверена техническими специалистами и не содержит вирусов или вредоносных модификаций.
Автоматизация: Программа выполняет действия, которые ранее выполнялись вручную, автоматизируя работу модераторов.

# Как скачать и установить
Для скачивания последней версии программы перейдите во вкладку Релизы на GitHub. Эти действия выполняет модератор, и вам не нужно вмешиваться в процесс загрузки.

# Какой файл скачивать
Скачайте сначала файл CordyChecker.exe, если программа не откроется, скачайте версию без уже встроеной JRE - CordyChecker-fixed.exe Иначе напишите в поддержку. Также если выдает ошибку каких-либо DLL файлов, скачайте пакеты C++

# Безопасность
CordyChecker был проверен техническим администратором ReallyWorld и не содержит вирусов или других вредоносных программ. Программа не требует доступа к вашим данным, она функционирует на основе четко прописанных алгоритмов, и её задача — только проверка и передача отчета администраторам.

# Разработка
Программа разрабатывалась в течение 8 месяцев, начиная с сентября, и прошла несколько версий. Изначально программа была написана на Python, однако позже было принято решение переписать её на Java и C++ для улучшения производительности.
Алгоритмы поиска были составлены лучшими модераторами СНГ, что гарантирует высокую точность работы чекера. Также стоит отметить, что Cordy Checker — это автоматизация действий, которые раньше выполнялись вручную модераторами.

# Рекомендации при проверке
Не вмешивайтесь в процесс: После подключения к вам модератора, не пытайтесь использовать компьютер, клавиатуру или мышь. Модератор имеет полную информацию о действиях, которые происходят в процессе проверки, и знает, что делать в каждом случае.
Используйте AnyDesk: Подключитесь к модератору через AnyDesk, чтобы он мог выполнить необходимые действия.
Удаление чекера: После завершения проверки Cordy Checker будет удален автоматически. Если по каким-то причинам это не произошло, и программа осталась на вашем компьютере, может быть применено наказание за отказ от проверки.
Результат: Модератор получил все результаты удобным ему способу, ваше взаимодействие не требуется...
